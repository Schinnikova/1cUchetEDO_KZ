# Выгрузка произвольных документов в Учет.ЭДО

Расширение на 1С конфигурации **"Управление торговлей для Казахстана, редакция 3"** для платформы 1С:Предприятие 8.3, предназначенное для интеграции с системой электронного документооборота [edo.uchet.kz](https://edo.uchet.kz).

## Описание

Расширение включает в себя обработку для отправки документов "Реализация товаров и услуг" из информационной базы 1С в систему Учет.ЭДО с электронной подписью, а также управления их статусами и отзыва отправленных документов.

## Возможности

- Отправка документов в Учет.ЭДО с электронной подписью
- Отзыв ранее отправленных документов
- Обновление статусов документов
- Проверка подключения к сервису
- Проверка настроек ЭЦП
- Открытие документов на сайте edo.uchet.kz

## Зависимости

Расширение использует библиотеку **КоннекторHTTP** для выполнения HTTP-запросов к API Учет.ЭДО.

- **Репозиторий:** [vbondarevsky/Connector](https://github.com/vbondarevsky/Connector)
- **Назначение:** Предоставляет удобный интерфейс для работы с HTTP/HTTPS запросами, включая поддержку прокси-серверов, авторизации и SSL-соединений

КоннекторHTTP уже включен в состав расширения, дополнительная установка не требуется.
